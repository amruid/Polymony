# Polymony

Wie man das Repository hinzufügt:
1. Ordner suchen, in dem man das Repository machen will
2. Ordner in Konsole öffnen
```
git init
git remote add origin https://github.com/fichte-paulsen-lk/Polymony
git pull
```
